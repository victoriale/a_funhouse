# JoyfulHome-Frontend
Be sure to be in the `develop` branch
1. `npm install
      -g gulp-cli
      gulp-less
      gulp-concat
      connect-history-api-fallback
      browser-sync`
2. `npm i`
3. `gulp serve`

Less files will be compiled to: `dist/app/global/stylesheets/master.css`

Router fix:
https://github.com/BrowserSync/browser-sync/issues/204
